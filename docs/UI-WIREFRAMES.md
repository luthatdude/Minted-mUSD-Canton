## Component Library

All pages use shared components:
- `StatCard` - Metric display with icon, value, label, and color
- `TxButton` - Transaction button with loading state
- `PageHeader` - Title, subtitle, and badge
- `WalletConnector` - Wallet connection UI
- `ChainToggle` - ETH ↔ Canton switcher
